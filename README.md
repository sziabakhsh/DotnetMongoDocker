# 🚀 .NET 8 Web API with Docker & MongoDB

This project demonstrates how to build and run a .NET 8 Web API using Docker and MongoDB.

## 🛠 Tech Stack
- .NET 8 (Minimal API)
- Docker & Docker Compose
- MongoDB
- Swagger (OpenAPI)

## 📦 Features
- REST API with sample endpoint
- Docker multi-stage build
- MongoDB integration
- Swagger UI for testing

## ▶️ Run with Docker

```bash
docker compose up --build
